# Spec: Onboarding Tour

## ADDED Requirements

### Requirement: Interactive Tour
The application MUST be able to display an interactive tour that highlights specific UI elements.

#### Scenario: First Launch Tour
On first launch, a tour starts and highlights the "New Tab" button.

### Requirement: One-time Display
The tour MUST only be shown once per user unless reset.

#### Scenario: Persistence
User completes tour. Re-launching the app does NOT show the tour again.
